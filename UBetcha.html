UBetcha Application

UBetcha Web Application Developed with Ember

This project was developed in my Advanced Web Development Course, a graduate level course offered to select students.My class and I developed this course using EmberJS, HTML, JavaScript, and CSS. I specficially built out the backend database using MongoDB.After creating the database locally, I wanted to host the databse on a server and discovered how to use MongoDB Atlas, a cloud database offered through MongoDB.This application represents a betting application that incorporates creating a user account. After creating an account, users can add friends and join each others bets, a cool feature current betting applications do not have.
